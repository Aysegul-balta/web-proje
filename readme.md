#### **Git İş Akışı Web Projesi**

###### 

###### Proje Açıklaması

Bu proje, Yazılım Mühendisliği dersi kapsamında verilen Git/GitHub ödevi için oluşturulmuştur. 

Projenin amacı Git iş akışını uygulamalı olarak öğrenmek, branch tabanlı geliştirme modelini kullanmak ve ekip çalışması sırasında ortaya çıkabilecek merge conflict durumlarını deneyimlemektir.



###### Proje Türü

Basit bir HTML/CSS tabanlı web sitesi.



###### Ekip Üyeleri

\- Esma Ece Kördeve

\- Ayşegül Balta

\- Elaheh Kazemi



###### Kullanılan Teknolojiler

\- HTML

\- CSS

\- Git

\- GitHub



###### &nbsp;Ödevin Amaçları

\- Git'in temel komutlarını öğrenmek

\- Branch tabanlı geliştirme modelini uygulamak

\- Pull Request sürecini deneyimlemek

\- Merge conflict durumlarını oluşturmak ve çözmek

\- Fetch, pull, merge ve rebase kavramlarını anlamak

\- Takım içi iş akışı ile çalışma deneyimi kazanmak
